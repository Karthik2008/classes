# Installing Docker on MacOs

1. Go to:

    https://docs.docker.com/desktop/install/mac-install/

2. Open and Setup Docker.

    Application > Docker

3. Installing Ubuntu with Docker, run:
    ```
    docker run -it ubuntu bash
    ```
4. Check the Os Version, run:
    ```
    > cat /etc/*se
    
    NAME="Ubuntu"
    VERSION_ID="22.04"
